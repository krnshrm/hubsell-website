By the end of this article you will have contacts in a hubsell list, sourced either from a LinkedIn or Sales Navigator search or from a CSV file.

**Prerequisites:** for LinkedIn sourcing, LinkedIn connected through the hubsell plugin. For CSV sourcing, a file with column headers and either a LinkedIn URL or an email address in every row.

> You can source data into hubsell from LinkedIn search, Sales Navigator search, or a CSV import. hubsell removes contacts you already have, so add a tag on every import to keep track of what came in when.

## Source from a LinkedIn or Sales Navigator search

1. Go to **Lists**, click **Add data** and select **LinkedIn**.

   ![Create new list](https://colony-recorder.s3.amazonaws.com/files/2026-07-20/b7cebd5c-b7b3-4f54-bc20-5e833254f7d8/user_cropped_screenshot_74d8c5302ea44f9392d8d38ec54abd52_text_export.jpeg)

   ![Naming the new list](https://colony-recorder.s3.amazonaws.com/files/2026-07-22/ee91e169-e043-462d-96b6-60e7f5ef9de3/user_cropped_screenshot_74bfd3e91f0842d09bef066c72b0a2ce_text_export.jpeg)

3. Click **Next**.

   ![Next button](https://colony-recorder.s3.amazonaws.com/files/2026-07-20/79fd4972-faf3-4ef1-9592-14f609037e37/user_cropped_screenshot_a32e1b1d40844650b00e4b63aacd67fe_text_export.jpeg)

4. Select the user this data should be assigned to.

   ![Assigning the data to a user](https://colony-recorder.s3.amazonaws.com/files/2026-07-20/96b09ff6-61c6-4306-a0f8-02f37b6b0819/user_cropped_screenshot_638a8720e6d54867b38e7716badd8095_text_export.jpeg)

5. In LinkedIn or Sales Navigator, run the search that returns your target customers and copy the search URL.

   ![Copying the search URL](https://colony-recorder.s3.amazonaws.com/files/2026-07-22/56768d47-c567-4a34-9b27-840f13a0f268/user_cropped_screenshot_0a115485f33b431cb47435302b1300e4_text_export.jpeg)

7. Select the enrichment level you need.

   ![Enrichment level options](https://colony-recorder.s3.amazonaws.com/files/2026-07-20/b2871f71-18ed-4cc3-8529-af721f1bb13e/user_cropped_screenshot_dd57198de81d49e9880ffa7ac3eeac9e_text_export.jpeg)

8. Add a tag so you can find this batch later. Choose an existing tag or click **+** to create one.

   ![Choosing or adding a tag](https://colony-recorder.s3.amazonaws.com/files/2026-07-20/aaf8748a-5499-480a-b994-269cd850ee90/user_cropped_screenshot_29b68ed0fa124b43978b91a95d4ee54d_text_export.jpeg)

   ![Flow assignment options](https://colony-recorder.s3.amazonaws.com/files/2026-07-20/a43a9cd0-5a68-4af3-8896-dc8c44a25b31/user_cropped_screenshot_e8216b8c57f64eab89462458941a2496_text_export.jpeg)

10. Click **Submit**, then click **Finish**.

## Source from a CSV import

13. Go to **Lists**, click **Add data** and select **CSV**.

    ![Add data menu with CSV selected](https://colony-recorder.s3.amazonaws.com/files/2026-07-22/7f5ded0d-129d-46d3-8e49-c5cd7e301d93/screenshot_5c4ca7e53cfb4e778133f0f12ead59ad_text_export.jpeg)

14. Select an existing list or create a new one, then click **Next**.

    ![Selecting a list for the import](https://colony-recorder.s3.amazonaws.com/files/2026-07-20/ce6d906a-266f-4f43-bddb-5e6e1412ba86/user_cropped_screenshot_546125c7007a4d10a61b42481da48604_text_export.jpeg)

15. Check your file before you upload it. Every column needs a header, and every row needs either a LinkedIn URL or an email address. If any names contain characters such as é, á or ó, save the file as CSV UTF-8.

    ![CSV requirements](https://colony-recorder.s3.amazonaws.com/files/2026-07-22/dd54d25e-6566-4114-98d3-26879630425d/user_cropped_screenshot_b20447b6f9d449f18b3d6c5d683b610f_text_export.jpeg)

16. Upload the file and click **Next**.

    ![Uploading the CSV](https://colony-recorder.s3.amazonaws.com/files/2026-07-20/455a0386-a365-4ead-ba82-c695e5364771/user_cropped_screenshot_ad08379f3dca40f493597a196298cb99_text_export.jpeg)

    ![Next after upload](https://colony-recorder.s3.amazonaws.com/files/2026-07-20/0a09eec2-7f00-450c-83af-7e42cd743a71/user_cropped_screenshot_2270ed0d02f94b9b8a2dc29a1ba0d9ee_text_export.jpeg)

17. Map the columns in your file to hubsell fields, then click **Next**. Anything without a matching hubsell field can be mapped to **Custom Variables**.

    ![Field mapping screen](https://colony-recorder.s3.amazonaws.com/files/2026-07-22/61165bb7-3376-43ed-9ddd-1e6237136705/user_cropped_screenshot_7d3edb71cc7b4909bc3918dd0d9ddac8_text_export.jpeg)

18. Choose an enrichment type, or skip enrichment.

    ![Enrichment options for CSV import](https://colony-recorder.s3.amazonaws.com/files/2026-07-20/e0b8e51f-82c0-404d-9ebc-80704feb4bce/user_cropped_screenshot_e46ad4413d704d4ca4804c85bfcaea66_text_export.jpeg)

19. Choose whether the contacts should go into an existing flow.

## What you should see

Your list appears under **Lists** with a count in the **Delivered** column, and the matching entry under **Searches** moves from **Active** to **Completed**.

Timings to expect:

- Background deduplication on a LinkedIn or Sales Navigator search takes 15 to 20 minutes, depending on list size.
- CSV deduplication takes roughly 20 to 30 minutes per few hundred contacts.
- Sales Navigator email enrichment finishes by the following day.

## Troubleshooting

- **Fewer contacts arrived than there were rows in the file.** This is expected. hubsell removes contacts you already have and skips rows with a broken LinkedIn URL or a missing mandatory field.
- **You cannot tell which contacts came from this import.** Tag every import. Without a tag, deduplication makes a batch hard to identify afterwards.
- **The CSV import failed.** Check that every column has a header and that every row has a LinkedIn URL or an email address.
- **Names show as garbled characters.** Excel can strip the encoding. Save the file as CSV UTF-8 and import again. If the contacts are already in, run LinkedIn enrichment to overwrite the corrupted names with the correct profile data.
- **Flow assignment fails on the CSV import screen.** Skip flow assignment during the import. Wait about 30 minutes, filter **Contacts** by list or tag plus **Status = New**, select all, then use **Add to flow**.
- **A search is stuck on Paused.** The LinkedIn or Sales Navigator session has gone out of sync. Disconnect and reconnect LinkedIn from the hubsell plugin.
- **Contacts entered a flow before you had checked them.** Suppress contacts during LinkedIn or Sales Navigator sourcing so poor fits are removed before they reach a flow.
