## Warum es wichtig ist

Eine Spam-Trap zu treffen ist ein starkes Signal, dass Ihre Liste gekauft, gescrapt oder schlecht gepflegt ist, und kann Ihre Domain schnell auf eine Blocklist bringen. Weil Traps wie gewöhnliche Adressen aussehen, können Sie sie nicht mit bloßem Auge erkennen. Sie zu vermeiden läuft auf gute Listenhygiene hinaus, weshalb Verifizierung und saubere Recherche so wichtig für den Schutz der Deliverability sind.

## Wie es in der Praxis funktioniert

Es gibt zwei Hauptarten. Eine reine Trap ist eine Adresse, die nur zum Erwischen von Absendern erstellt wurde und sich nie für etwas angemeldet hat, sodass jede Mail an sie bedeutet, dass die Adresse gescrapt oder geraten wurde. Eine recycelte Trap ist eine alte verlassene Adresse, die ein Anbieter als Trap reaktiviert, sodass ein Treffer bedeutet, dass Ihre Liste veraltet ist. Beide werden still über das Web verteilt.

## Häufige Fehler

Der Grundfehler ist, Listen zu kaufen oder zu scrapen, so gelangen Traps in eine Datenbank. Ein weiterer ist, alte Datensätze nie zu bereinigen, so erwischen Sie recycelte Traps. Ein dritter ist, steigende Bounce- und Beschwerdesignale zu ignorieren, die oft eine Liste voller Traps begleiten.
